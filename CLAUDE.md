# CLAUDE.md - uscpaプロジェクト

## 基本ルール
- 作業開始時に必ずgit pullを実行する
- 新しいファイルを作成する前に必ず「〇〇を作成してもよいですか？」と確認する
- コンテキスト使用量が50%を超えたら/compactを実行する

## 保存・同期ルール
- ユーザーが「OK」「いいね」「今日はここまで」「ありがとう」などの区切りの言葉を言ったタイミングでcommit & pushする
- 決定事項・設計の理由・方針変更があれば都度このCLAUDE.mdに追記する
  その際、変更前の内容を消さず「→変更」の形で経緯も残すこと
  追記後はそのままcommit & pushする

## プロジェクト概要

USCPA試験の問題演習Webアプリ。「やりながら覚える」学習スタイルに合わせ、無限に問題を解ける環境を作る。

## ブレスト議事録（2026-02-21）

### 背景・課題（ChatGPTで演習していた際の痛み）
- AIが生成する問題の品質が低い（消去法で解けてしまう）
- 計算問題の選択肢に正解が存在しないことがある
- 誤答時にヒントではなく正解を教えられてしまう
- ChatGPT → Gemini → Notionへの手動コピペが面倒

### 問題ソース方針
- **メイン：** abitusテキスト（docx形式、購入済み）を種として使用
- **補助：** AICPA公式リソース（Sample Questions、Exam Blueprints）
- **生成：** docxを根拠としてClaudeが類似・応用問題を無限生成
- ネットからの自動収集は技術・著作権の面でリスクが高いため対象外

### 出題タイプ（4種）
1. 読み込んだ問題をそのまま出題
2. 読み込んだ文章問題の選択肢を変更して出題
3. 読み込んだ計算問題の数字と選択肢を変更して出題
4. AIが作成したオリジナル問題

### 学習ロジック
- 単元ごとに10問連続正解 → その単元を10日間スキップ（間隔反復法）
- 誤答時：正解は教えず、ヒント（考え方・着眼点）を1つだけ提示
- 誤答した問題は同一単元の1問目からやり直し（問題・選択肢は変更）

### 計算問題の品質担保（重要課題）
- **2段階検証方式：** 問題生成（1回目のAPIコール）→ 自動検証（2回目のAPIコール）→ NGなら再生成 → 検証済みのみユーザーに出題
- ユーザーには検証済みの問題だけ表示されるのでシームレス

### 出題モデル（確定：ハイブリッド）
→変更：「次回決定」→「ハイブリッドに確定」（2026-02-22）

- **ハイブリッド方式**を採用（品質重視）
- 生成 → 2段階検証 → Sheetsに保存 → 出題（APIコスト不要）
- **問題プールの管理：定期再生成（GASスケジュール実行）**
  - 毎週自動でSheetsの正誤データを読み、苦手傾向を考慮して問題を補充
  - ユーザーは何もしなくていい（月曜朝には新問題が入っている状態）

### データ階層構造（確定）
```
科目（FAR / AUD / REG ...）
  └ 単元（Chapter単位）
       └ 問題プール（常時100問程度を維持）
            └ 各問題：設問・正解・ヒント・種別・難易度・正誤履歴
```

### 問題への付与情報（確定：シンプル版）
→変更：概念タグ案を検討したが「タグ設計に時間を取られる・本質的でない」として不採用
- **種別：** vocab（用語）/ calculation（計算）のみ
- **難易度：** easy / medium / hard のみ
- 再生成時に種別・難易度の比率を維持する指示をClaudeに渡す

### 正誤履歴の管理（確定）
- 各問題：正誤履歴を記録（再生成でリセット）
- 単元レベル：直近100問の正誤履歴を継続蓄積（問題の入れ替えをまたいで継続）
- 科目レベル：単元の集計から習熟度を観測

### 技術スタック（確定）
- **フロントエンド：** HTML/CSS/JS（レスポンシブ、スマホ対応）
- **バックエンド：** Google Apps Script（GAS）
- **AI：** Claude API（用語問題→Haiku、計算問題→Sonnet）
- **DB：** Google Sheets（問題DB・進捗・誤答ログ）

### コスト感
- 1問あたり：Haiku〜$0.003、Sonnet〜$0.015
- 通常運用（30問/日）：月1,000〜2,000円程度
- フルブースト（200問/日×30日）：月2,700〜13,000円程度

### 科目・目次
- 対象：全科目（AUD / FAR / REG / BAR / ISC / TCP）
- 学習開始科目：FAR
- 目次の正：abitusテキストの目次をベースに、AICPAのExam Blueprintsと対応付け
- ※abitusの目次は次回コピペして共有予定
